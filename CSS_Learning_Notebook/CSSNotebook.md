### 02 css是什么  

- css(Cascading Style Sheets),层叠样式表，是一种_样式表_语言，用来描述 HTML 或 XML文档的呈现。css不是编程语言，css描述了在屏幕、纸质、音频等其它媒体上的元素应该如何被渲染的问题。
