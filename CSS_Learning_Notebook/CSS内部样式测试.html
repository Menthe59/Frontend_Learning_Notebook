这是一个测试盒子，id为innerTest,用于测试内部样式，将内部字体变为蓝色。一般来说不使用这种方法。
当想让一个元素显示两种css属性时，可以在class用空格隔开两个.后的名称。
id最好在整个页面中唯一，当想为同一类元素/div设置同种样式时，只需给标签添加相同class属性，并且在style内用“.”引用。
例如为这个div添加粉色背景，同时让字体显示为红色。
此div设定为背景颜色为粉色,字体颜色默认。
"#"称为id选择器，"."称为class选择器。还有“标签选择器”，可以直接对某一类标签添加效果，写法为div{}。
例如，为现有的三个div添加宽度5px的蓝色点阵边框，这三个属性可以分开写也可以一次写完。
