元素默认有块级block和内联inline两种属性分类，其最大的区别在于块级元素其后的元素会另起一行而内联元素不会。
而css的display属性可以用来改变元素的这“内联”、“块级”属性。
display的取值表
值 描述
none 此元素不会被显示，即隐藏
block 此元素将被显示为块级元素
inline 默认，此元素会被显示为内联元素，失去宽度高度
inline-block 行内块元素，即具有宽度、高度，但处于行内的，不会独占一行（css2.1新增的值）
去除间距的n种方法请点击其他取值请自行查找

例如，span不是块级，所以其宽度是依据内容多少而定，或者必须设定span为block才可以设置宽度，宽度才会有效。
